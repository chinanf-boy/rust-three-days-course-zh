# 教 Rust

**三天就'生锈'(Rust)**

"Teach Rust"是一个免费的研讨会材料,用于介绍 Rust 编程语言的课程。完整课程的时间大约是三到四天.

这些材料是由具有零 Rust 经验，但具有编程背景的人创建的。

## 商业可用性

本课程由[asquera](http://asquera.de)赞助和培训，语言包括德语和英语。[问问客服邮件](mailto:info@asquera.de)关于定制报价。

西班牙语翻译由社区维护，我们很欢迎你加入。

## 概观

该课程由两部分组成:

[课程介绍](presentation)用作 Rust 的所有基本方面的介绍。它有小部件,并且与 Rust 游乐场集成，例如带有翻译结构。

[工作 实例](example)。该课程从头开始构建一个小型 TCP 服务器.

与此同时,我们使用的所有通信材料都是[这里](communication-material)提供的.

## 演讲

幻灯片材料分为许多个，中小型幻灯片文稿，关于 Rust 的各个方面。并非所有都是一个主题的完全涵盖，多数作为只做重点的介绍。而涵盖到所有权和借用等基本内容的章节，应该是详尽无遗的。

幻灯片文稿是用 Reveal（.js） 幻灯片文稿，其中包含用于国际化和集成[Rust Playground](https://play.rust-lang.org)的插件.

通过幻灯片的路径并不固定，以便在举办课程期间留有余地。我们使用的 pathes 的例子，可在[courses](courses)目录找到。我们建议您在准备参加研讨会时，创建自己的.

## 例子

目前,该课程附带三个例子:

- 一个简单的 TCP 服务器,提供一个非常简单的 PUT 和 GET 接口来存储和删除消息
- 使用 Tokio 和 Futures 的 相同(上一个)例子
- 可以用作动态语言的 一个 Rust 库

TCP 服务器应该从零开始构建，新箱的起点

这里给出的例子，都为最终状态。在开始本课程之前,您应该自己开发这个例子.

## 任务

可以在`src/bin/`找到各种任务，并且可以运行`cargo run --bin $TASK"`.

任务如下:

- `add-chapter $CHAPTER`- 按名称创建章节，其中`$CHAPTER`是每个(语言)区域设置.
- `add-code-example $CHAPTER $INDEX`- 适当地重命名所有现有代码示例(<=`$INDEX` + 1)，让`$CHAPTER`可以，为新的代码示例腾出空间。还会更新幻灯片以反映这一点。
- `build-indices`- 在`presentation/toc/`目录建立对应`md`，将`.md`变为`html`用于 Web 浏览器.
- `extract-code`- 从所有章节中的幻灯片中提取代码样本,并在`presentation/chapters/shared/code/*/*`中创建适当的副本,然后使用反映新位置的新元素更新幻灯片，不需要经常使用。
- `rename-chapter $OLD $NEW`- 重命名`$OLD`章节到`$NEW`，适用于所有语言环境.
- `rename-chapters`- 将 chapters 的`md`重命名到`chapter`后缀。不应该经常使用。
